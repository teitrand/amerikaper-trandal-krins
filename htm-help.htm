Æ     -    &AElig; becomes/ vert    - Æ ...(alternatively, one can use &#0198 to get Æ.)
æ     -    &aelig; becomes/ vert    - æ ...(alternatively, one can use &#0230 to get æ.)
Ø     -    &Oslash; becomes/ vert    - Ø ...(alternatively, one can use &#0216 to get Ø.)
ø     -    &oslash; becomes/ vert    - ø ...(alternatively, one can use &#0248 to get ø.)
Å     -    &Aring becomes/ vert    - Å ...(alternatively, one can use &#0197 to get Å.)
å     -    &aring; becomes/ vert    - å ...(alternatively, one can use &#0229 to get å.)

&     -    &amp; becomes/ vert    - & ...(alternatively, one can use &#38 to get & ampersand sign.)
"     -    &quot; becomes/ vert    - " ...(alternatively, one can use &#34 to get " quotation mark.)

Døme/ Example:
"Været kan sjå ut ...som snø & regn."
